# Witcher Character Network Analysis

A graph-based analysis of Witcher character relationships using Python and NetworkX.

## Overview

This project represents Witcher characters as a network.

- Characters = nodes
- Relationships = edges
- Relationship strength = edge weight

The goal is to identify important characters, strong relationships, communities, and book-wise relationship patterns.

## Tools Used

- Python
- Pandas
- NetworkX
- Matplotlib

## Analysis Performed

- Cleaned and combined repeated undirected relationships
- Built a weighted character network
- Found most connected characters
- Calculated relationship strength
- Used centrality measures to identify important characters
- Detected character communities
- Analyzed relationship activity across books

## Key Findings

- Geralt is the main hub of the network.
- Ciri, Yennefer, and Dandelion are also important characters.
- The strongest relationships are centered around main characters.
- Community detection shows smaller character groups within the network.
- Book-wise analysis shows how relationship activity changes across books.
